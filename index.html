Workflow

  • Project

    Whole process starts with an idea, a concept which become live on peace of paper - oldschool but it is convinient and much faster. Next step is to create mockup in Photoshop. One of many - mockup, another ideas, different solutions, another mockup till we find exactly what you are looking for.

  • Tools & Settings

    At this point I create an enviroment for your future website. I use Gulp, Git, Jekyll - If required. Sass wich is very convinient tool.
    All this preperations are for faster delivery what you want - your website

  • Coding

    Now is the time when real fun begins. When everything is set up, goals are clear it is time to write some code. First lines of HTML are written, then CSS and some JavaScript for more interaction to keep your users on your website - they eventually press that "BUY" button.

  • Quality Assurance

    When everything is done it is time to check if everything works great. I take care of responsiveness simultaneously while I am coding different sections but it is a good time to check it again... and again.
    Untill everything will be exactly what you wanted
